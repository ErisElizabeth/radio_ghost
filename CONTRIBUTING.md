# Contributing

Radio Ghost is early and changing quickly.

## Local Setup

```powershell
npm install
npm start
```

## Checks

Run this before committing:

```powershell
npm run check
```

## Pull Requests

- Keep changes focused.
- Describe user-facing behavior changes.
- Include before/after notes for audio behavior when relevant.
- Avoid committing generated app builds from `release/`.
